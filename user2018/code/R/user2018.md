================
Neil Saunders
2018-07-16 21:55:35

Introduction
============

An analysis of tweets from the useR 2018 meeting. 5967 tweets were collected using the `rtweet` R package:

``` r
library(rtweet)
user2018 <- search_tweets("#useR2018", 10000)
```
