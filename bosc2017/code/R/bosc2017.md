Neil Saunders
2018-07-16 22:40:09

Introduction
============

An analysis of tweets from the BOSC 2017 meeting. 3268 tweets were collected using the `rtweet` R package:

``` r
library(rtweet)
bosc17 <- search_tweets("#bosc2017", 5000)
```
